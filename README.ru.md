# Bad Cut - Вымышленный интернет-магазин кепок

Учебный проект интернет-магазина на чистом PHP с использованием MySQL.

## Технологии

- Backend: PHP 7.4+
- База данных: MySQL
- Расширение: MySQLi
- Frontend: HTML5, CSS3, JavaScript
- Безопасность: Prepared Statements, сессии, хеширование паролей

## Архитектурные решения
- Модульная структура - разделение на переиспользуемые компоненты (header, footer)
- MySQLi с Prepared Statements - защита от SQL-инъекций
- Сессии PHP - управление состоянием пользователя
- Древовидное меню и комментарии - многоуровневые системы через `parent_id`
- Валидация на стороне сервера - проверка всех входных данных
- Обработка ошибок - вывод сообщений об ошибках пользователю

## Реализованные возможности
### Система авторизации
- Регистрация с валидацией данных (логин, email, телефон, пароль)
- Проверка на дубликаты - нельзя зарегистрироваться с существующим логином/email/телефоном
- Безопасное хранение паролей - хеширование через MD5
- Сессии - хранение данных пользователя между страницами
- Авторизация с проверкой учётных данных
- Выход из системы

### Каталог товаров
- Динамическая генерация каталога из базы данных
- Фильтрация по брендам - выбор конкретного производителя
- Фильтрация по категориям - группировка товаров по типу
- Сортировка:
  - Сначала дешёвые
  - Сначала дорогие
  - По алфавиту (А-Я, Я-А)
- Сброс фильтров одной кнопкой

### Страница товара
- Детальная информация о товаре (название, описание, цена, изображение)
- Система комментариев:
  - Добавление отзывов от авторизованных пользователей
  - Ответы на комментарии (древовидная структура)
  - Удаление своих комментариев
- Галерея изображений товара
